The artificial neuron

  • Inspired by biological neurons
  • Each layer in network:
    • a collection of neurons
  • Each neuron: $o = a(w*x+b)$
  • $w, b$: trainable parameters
  • A pattern as input
  • A value (scalar) as output

Example AI tasks

for environmental data

  • Environmental monitoring
    • Species identification from camera traps / acoustic monitoring
    • Forest loss detection from satellite data
    • Carbon stock mapping using remote sensing
  • Prediction
    • Short-term: Flood risk mapping, drought forecasting
    • Long-term: Climate model downscaling, ecosystem resilience projections
  • Knowledge / Search
    • Mining scientific literature for climate-relevant knowledge
    • Aggregating heterogeneous environmental datasets
  • Decision support
    • Urban planning for climate resilience
    • Optimizing locations for restoration / renewable energy
    • Early warning systems for extreme events
  • Estimating emissions
  • Optimizing activities that create GHG emissions (e.g. logistics, transport, industry)
